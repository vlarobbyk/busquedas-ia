Representación de problemas y estrategias de solución

En esta unidad se abordarán los principales conceptos relacionados con la representación de problemas en la Inteligencia Artificial (IA), la representación del conocimiento, los métodos de búsqueda "sin razonamiento" y los métodos de búsqueda "heurísticos".

Requisitos de software

A fin de poder trabajar de forma más eficiente, se sugiere instalar la plataforma científica de datos Anaconda. Luego de ello, deberemos instalar algunos paquetes que nos permitirán desarrollar los algoritmos de búsquedas y ejemplos de resolución de problemas de satisfacción de restricciones (a fin de ilustrar de forma práctica cómo se representa el conocimiento para cierto tipo de problemas).

A continuación se describen los pasos para realizar la instalación de las herramientas requeridas. Se sugiere empelar cualquiera de las distribuciones de GNU/Linux (Ubuntu 16 LTS, KUbuntu 16 LTS, etc.), ya que algunos paquetes no son sencillos de instalar en Windows.

  1. Instalación de Anaconda
    • Como primer paso debemos descargar el instalador desde la página oficial: Descargas (preferible la versión para Python 3.7).
    • Una vez descargado, deberemos ubicarnos en la carpeta donde se almacenó el fichero (Anaconda3-5.3.0-Linux-x86_64.sh) y ejecutar el siguiente comando: 
      • sudo bash Anaconda3-5.3.0-Linux-x86_64.sh
    • Deberemos aceptar la licencia e indicar la ubicación donde deseamos que se instale Anaconda.
    • En el siguiente vídeo se puede observar el proceso para distribuciones basadas en Ubuntu 16.04 (LTS):
  2. Creación de entornos (no es necesario ejecutar este paso, ya que el paso siguiente se encarga de crear el entorno e instalar los paquetes requeridos): 
    • Con el siguiente comando procedemos a crear los entornos que se requieran:
    • conda create --name iaups

      • Es importante tomar en cuenta que también podemos pedir que nos instale paquetes cuando se crea el entorno, por ejemplo que se cree el entorno con la versión 3.7 de Python:
      • conda create --name iaups python=3.7
      • Si hemos cometido algún error con la creación del entorno, podemos borrarlo fácilmente con el siguiente comando:
      • conda env remove --name iaups
  3. Instalación de librerías requeridas
  • A fin de poder resolver los diversos problemas y ejercicios que se plantearán a lo largo de la asignatura, se sugiere instalar los siguientes paquetes:
    • cython=0.28.2
    • matplotlib=2.1.1
    • jupyter=1.0.0
    • nltk=3.2.5
    • nb_conda=2.2*
    • pandas=0.22.0
    • pomegranate=0.9.0
    • pydot=1.2.4
    • pydotplus=2.0.2
    • python=3.5.4
    • scikit-learn=0.19.1
    • scipy=1.0.0
    • tqdm=4.19.4
    • z3-solver
  • Sin embargo, para facilitar la tarea, en el AVAC encontrará un archivo llamado ia1-software.yml que contiene el descriptor para instalar todas las librerías  y crear el entorno iaups. Deberemos descargar el archivo y ejecutar el siguiente comando: 
    • conda env create -f ia1-software.yml
      Dicho comando realizará de forma automática la instalación de las herramientas requeridas y la generación del entorno. Finalmente, se sugiere instalar la herramienta Pygame (en el entorno iaups que se ha creado), de la siguiente forma:
    • conda install --name iaups -c cogsci pygame
      El parámetro "-c cogsci" indica que la librería pygame debe buscarse en el canal (repositorio) "cogsci".
    • Con ello, estamos listos para poner en marcha nuestro entorno y usar Python:
    • // Para activar el entorno usamos:
source activate iaups 
// Luego podemos acceder a la consola interactiva
python

// Para desactivar el entorno (una vez que salimos de Python con el comando exit()):
source deactivate

// Si deseamos correr un script desarrollado en Python, podemos ejecutar los siguientes comandos:

source activate iaups
python Fichero.py
